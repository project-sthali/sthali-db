PaginateParameters

Represents the parameters for retrieving items.

Attributes:
    skip (pydantic.NonNegativeInt): The number of items to skip. Defaults to 0.
    limit (pydantic.NonNegativeInt): The maximum number of items to return. Defaults to 100.
